# MinhasHoras Documentation

Available at
https://minhashoras.com.br/docs

## Building

First install the following from pip:

* mkdocs
* mkdocs-material
* pymdown-extensions

Then simply run `mkdocs serve` to host a local version of the docs, or `mkdocs
build` to output a static version.
